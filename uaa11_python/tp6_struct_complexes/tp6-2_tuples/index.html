          >>> tpl = ...

          >>> tpl = ...

    Découv 2 : Afficher des éléments d'un tuple

    1. Qu'affiche la dernière ligne du code suivant ? 

          >>> tpl = (1,8,7,-5)
    >>> print(tpl[1])

    2. Complète le code ci-dessous pour qu'il affiche la troisième valeur du tuple tpl définie à la première ligne. 

          >>> tpl = (1,5,8,7,9)
    >>> print(tpl[...])

    Découv 3 : Modifier des éléments d'un tuple

    1. Qu'affiche le code suivant ? 

          >>> tpl = (1,5,7,-5,9)
    >>> tpl[3] = 0
    >>> print(tpl)

    >>> ...
