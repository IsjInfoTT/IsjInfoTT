    3. Trouve l'instruction permettant de connaître la taille de la variable jour. Quelle est-elle ?

    4. Attention, la numérotation de ces index commence à partir de 0.

      A quels éléments permettent d'accéder les instructions suivantes ? 

          >>> jour[1]
    >>> jour[3]

    6. Quelle est l'instruction permettant d'accéder au dernier élément de la liste jour ?

    2. Complète le code suivant de telle sorte que la liste lst définie à la première ligne devienne [1,5,4,0,8].

    3. Complète le code suivant de telle sorte que la liste lst définie à la première ligne devienne [1,-5,3,4,2]. 
          >>> lst = [1,5,3,4,2]
    >>> lst[...] = -5
    4. Complète le code suivant de telle sorte que la liste lst définie à la première ligne devienne [1,0,3,5,2].

    2. lst[0] = 1
